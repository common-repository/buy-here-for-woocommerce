Connect WooCommerce products to physical locations managed by Store Locator Plus.

== Description ==

Extend WooCommerce to connect products to [Store Locator Plus](https://www.storelocatorplus.com/) managed locations.

This initial release provides a way to show which products are available at your locations.   

Future updates as well as premium add-ons for Store Locator Plus are in development that will provide more interactive searches and listings between products and physical locations.

https://www.youtube.com/watch?v=94Z3LSpdEgg

= WooCommerce Features =

Locations are assigned using a WooCommerce-standard product details interface that lists-and-attaches locations from the Store Locator Plus locations table.
Locations attached to products are listed under the WooCommerce product details.

= Store Locator Plus Features =

Adds a [slp_location products.list] shortcode option that can be used in the Results Layout and Info Bubble layout provided by the [Experience add-on](https://www.storelocatorplus.com/product/experience/).

== Installation ==

= Requirements =

* [Store Locator Plus](https://www.storelocatorplus.com/): 4.4.18
* WooCommerce 2.4.10
* WordPress: 3.8
* PHP: 5.2.4

= Requires Store Locator Plus and WooCommerce =

* Install the latest version of [Store Locator Plus from the WordPress Plugin Directory](https://wordpress.org/plugins/store-locator-le/).
* Install the latest version of [WooCommerce from the WordPress Plugin Directory](https://wordpress.org/plugins/woocommerce/).
* Add a product in WooCommerce and link it to a Store Locator Plus location.
